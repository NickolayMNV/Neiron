******************Инструкция по запуску********************
Программа сформирована в exe-файл face.exe. В штатном режиме программа должна запуститься без ошибок при двойном нажатии на файл. 
***********************************************************
Настоятельно рекомендую проверить веб-камеру. Если она не установлена, программа работать не будет, если их несколько, то узнайте id
желаемой для запуска веб-камеры. по умолчанию в программе установлено 0. Если что-то пошло не так, например, вам нужно поменять id на 1, 2 и так далее, вам потребуется python-файл face.py. Во второй строчке кода cap = cv2.VideoCapture(0) значение 0 изменить на id вашей камеры.
Повторюсь, этого делать не нужно, если у вас одна рабочая веб-камера.
